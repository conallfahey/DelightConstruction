# 4) On‑Page SEO Best Practices

- One **H1** per page; descriptive **H2/H3** with keywords naturally.
- Geo‑modifiers in titles, H1s, and body: *Missoula*, *Seely Lake*, *Western Montana*.
- Short, human meta descriptions (≤155 chars) with a verb + location.
- Descriptive alt text for all images (room type, material, city).
- Internal links: Every service + city page links to 2–3 related pages and the Contact page.
- Use project pages to target long‑tail queries (materials, styles, neighborhoods).

# 5) Schema (JSON‑LD)

Embed via script tag on relevant pages.

**Sitewide LocalBusiness (on Home):**

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Delight Construction",
  "image": "https://example.com/images/og-cover.jpg",
  "url": "https://delightconstruction.com/",
  "telephone": "+1-406-000-0000",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Missoula",
    "addressRegion": "MT",
    "addressCountry": "US"
  },
  "areaServed": ["Missoula MT", "Seely Lake MT", "Bonner MT", "Deer Lodge MT", "Western Montana"],
  "priceRange": "$$$$",
  "openingHours": "Mo-Fr 08:00-17:00"
}
</script>
```

**Service Page (Custom Homes):**

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Service",
  "serviceType": "Custom Home Building",
  "provider": {
    "@type": "LocalBusiness",
    "name": "Delight Construction"
  },
  "areaServed": {
    "@type": "AdministrativeArea",
    "name": "Missoula, Western Montana"
  }
}
</script>
```

**FAQ (on Financing & FAQ):**

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
```

# 10) Technical SEO & Performance

- Fast hosting, HTTP/2 or HTTP/3, Brotli/Gzip.
- Image optimization (WebP/AVIF, lazy‑load, responsive sizes).
- Semantic HTML, ARIA labels, accessible contrast.
- XML sitemap, robots.txt, canonical tags.
- 301s for any slug changes; clean URL slugs with city/state where relevant.
- Core Web Vitals targets: LCP < 2.5s, CLS < 0.1, INP < 200ms.

---

# 11) Analytics & Tracking

- GA4 + Google Tag Manager.
- Goals: Form submits, phone clicks, email clicks, file downloads.
- Events: Hero CTA, sticky header CTA, portfolio image expand, map clicks.
- Call tracking number (local area code) mirrored in GBP and site NAP.
